Relocation Patterns in SF (work in progress)

7. December 2015

This is a visualization inspired by the Anti-Eviction Mapping Project, a data-visualization and digital storytelling collective working to document the housing crisis in the San Francisco Bay Area in California.

Click the image to view.

About

Each link in this graph represents people relocating from one zip code within San Francisco to another. The width of the link represents the number of people moving. Red links correspond to destination zip codes and darker blue links correspond to source zip codes. Data provided by the Anti-Eviction Mapping Project. This is a work in progress!
